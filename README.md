## Settung Up

First Install Ruby Version Manager
then run

```bash
rvm use
gem install bundle
bundle install
bundle exec jekyll serve -w --config _config.yml,_config-dev.yml
```

## how to update CSS and JS

Install Node Version Manager (NVM)

```bash
nvm use
npm i
npm run build
```
